INTRODUCCIÓN



El presente OVI se ha desarrollado con el proceso de aprendizaje adquirido al momento del curso de diseño de sitios web, mediante el cual se ha desarrollado una pagina web, de acuerdo a un diseño o mockup planteado previavemte por el grupo.

Con este trabajo se explicarán temas relacionados con la creacion de un software y los pasos que se requieren para completarlo.

Este tabajo es el proceso de desarrollo de los participantes del curso que mediante el proceso de investigacion y de aprendizaje de un sitio web con el uso de la herramienta de Github otorgando permisos de edicion y creacion para la posterior validacion del producto final.
