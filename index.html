INTRODUCCIÓN


El presente desarrollo OVI, se ha desarrollado con el proceso de aprendizaje adquirido al momento del curso de diseño de sitios web, mediante el cual se ha desarrollado una pagina web, de acuerdo a un diseño o mockup planteado previavemte por el grupo, con este trabajo se explicaran temas relacionados en como se lleva a cabo la creacion de un software y de los pasos que se requieren para completarlo. Este tabajo es el proceso de desarrollo de los participantes del curso que mediante el proceso de investigacion y de aprendizaje se ha desarrollado un sitio web mediante la herramienta de github otorgando permisos de edicion y creacion para la posterior validacion del producto final.
